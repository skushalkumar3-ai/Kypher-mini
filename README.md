## 📖 Overview

**Kypher Mini** is a lightweight desktop AI companion built using **Electron**, **JavaScript**, **HTML**, and **CSS**.

Designed to stay minimal yet powerful, Kypher Mini provides an elegant desktop experience with AI interaction, voice capabilities, and productivity-focused utilities inside a modern cyber-inspired interface.

The goal of this project is to create a desktop assistant that feels natural, responsive, and visually appealing without consuming excessive system resources.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/skushalkumar3-ai/Kypher-mini.git
```

Go into the project folder

```bash
cd Kypher-mini
```

Install dependencies

```bash
npm install
```

Run the application

```bash
npm start
```
